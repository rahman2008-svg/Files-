Files-

A modern Android file management and productivity application built with Kotlin, Jetpack Compose, Material 3, and modern Android development practices.

---

📱 Overview

Files- is designed to provide a fast, beautiful, and user-friendly file management experience for Android users. The application focuses on performance, simplicity, privacy, and modern UI design.

---

✨ Features

- Modern Material 3 Design
- Jetpack Compose UI
- Fast File Browsing
- Media Management
- Storage Analysis
- Clean Architecture
- Offline Functionality
- Smooth User Experience
- Optimized Performance
- Privacy-Focused Design

---

🛠 Technology Stack

- Kotlin
- Jetpack Compose
- Material 3
- Android SDK
- Coroutines
- Room Database
- Retrofit
- Coil
- Firebase Services
- Navigation Compose

🚀 Getting Started

Clone Repository

git clone https://github.com/rahman2008-svg/Files-.git
cd Files-

Build Debug APK

chmod +x gradlew
./gradlew assembleDebug

Build Release APK

./gradlew assembleRelease
